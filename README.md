# THRIVEair Shiny App
This code produced an interactive online dashboard for a community-based volatile organic compound (VOC) air monitoring study in S/SW Philadelphia called THRIVEair. Data collection ran from June 2023-June 2024. The app can be accessed at [this link](https://lisa-frueh.shinyapps.io/THRIVEair_results/).

### About THRIVEair

THRIVEair is a community-based air monitoring project in South and Southwest Philadelphia. Designed as a partnership between [Philly Thrive](https://www.phillythrive.org/) and researchers at the [Drexel Dornsife School of Public Health](https://drexel.edu/dornsife/) and [Lewis Clark State College](https://www.lcsc.edu/). Data will be collected from summer 2023 - summer 2024.

**For more information on the study, visit our website at: [THRIVEairPhilly.com](https://thriveairphilly.com/)**
